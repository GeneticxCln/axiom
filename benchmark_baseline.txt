# Axiom benchmark baseline notes
#
# This repository uses Criterion's built-in baseline support instead of
# maintaining a hand-written CSV in git.
#
# Create a local baseline:
#   bash ./scripts/benchmark.sh save-baseline local-main
#
# Compare a later run against that baseline:
#   bash ./scripts/benchmark.sh compare local-main
#
# Raw benchmark artifacts and comparison reports are written under:
#   target/criterion/
#
# The CI performance job currently records benchmark artifacts for inspection,
# but does not claim an automatic pass/fail regression threshold yet.
